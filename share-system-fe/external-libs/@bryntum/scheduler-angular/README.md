# Angular wrapper for Bryntum Scheduler
This package contains the wrapper that encapsulates the Bryntum Scheduler and turns it into Angular component exposing configuration options, properties, features and events.

## Installation
The installation is performed by issuing the installation command in the root of the application folder.
The specific command depends on the package manager used by the application.
The following are most frequently used:

Install using `npm`:
```
npm install @bryntum/scheduler-angular
```
Install using `yarn`:
```
yarn add @bryntum/scheduler-angular
```

## Dependencies

@bryntum/scheduler

To get more information on the package installation and use, use [Angular Integration Guide](https://www.bryntum.com/docs/scheduler/#guides/integration/angular.md)
